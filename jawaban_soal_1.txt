Git conflict terjadi ketika dua branch memiliki perbedaan di file sama sehingga Git tidak bisa menggabungkan kedua branch tersebut secara langsung. Cara menuntaskan sebuah git conflict ada beberapa cara. Pertama, kita terlebih dahulu cari tahu konfliknya di mana. Lalu, kita buka file yang konflik tersebut dan kita ubah kode yang konflik tersebut. Setelah beres, kita simpan lalu kita lanjutkan proses merging kita.
