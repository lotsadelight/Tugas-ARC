Fork (kalau diterjemahkan artinya percabangan, bukan garpu) adalah salinan dari sebuah repository. Ketika sebuah repository difork, kita bisa dengan leluasa memanipulasi repository tersebut sesuai dengan keinginan kita. Setelah melakukan berbagai perubahan yang kita inginkan, kita bisa meminta pemilik repository asli untuk memasukkan perubahan-perubahan yang telah kita buat dengan membuat sebuah pull request. Pull request adalah sebuah cara untuk meminta pemilik repository supaya menggabungkan branch fork tersebut ke main.
